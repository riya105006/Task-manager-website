## Project Note

This project was built as part of my web development learning journey using HTML, CSS, and JavaScript.

The main goal of this project was to practice DOM manipulation, event handling, form validation, localStorage, and building a complete frontend application from scratch.

The project is functional, but it is a learning project rather than a production-ready application. Some features may have limitations, and there are known issues that have not yet been addressed.

This repository represents my understanding and progress at the time it was built.
